# Project 3: Contact Form

## Learning Objectives

### HTML Concepts:
1. **Forms**: Create a form using `<form>` element
2. **Form Elements**: Use `<input>`, `<textarea>`, `<select>`, and `<button>`
3. **Form Attributes**: Add type, placeholder, required attributes
4. **Labels**: Properly associate labels with form inputs for accessibility

### CSS Concepts:
1. **Pseudo-classes**: Style `:hover`, `:focus`, and `:active` states
2. **Transitions**: Add smooth transitions for interactive elements
3. **CSS Variables**: Use custom properties for consistent theming
4. **Form Styling**: Create beautiful, modern form designs

## Instructions

1. Create form structure in `index.html`:
   - Add a form with name, email, subject, and message fields
   - Use appropriate input types (text, email)
   - Add labels for each input
   - Include a submit button
   - Add required attributes where needed

2. Style the form in `styles.css`:
   - Define CSS variables for colors
   - Style input fields with borders and padding
   - Add hover and focus states with transitions
   - Style the submit button with hover effects
   - Make the form visually appealing

## Expected Result
A beautiful contact form with:
- Properly labeled inputs
- Smooth hover and focus effects
- Consistent color scheme using CSS variables
- Professional appearance
